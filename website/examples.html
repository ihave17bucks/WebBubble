1. REST API with Database

A complete CRUD API in under 30 lines. Perfect for MVPs and microservices.

// Complete REST API with SQLite route "/api/users" { users = database.query("SELECT * FROM users") response json { users } } route "/api/users/:id" { user = database.get("users", id) if user { response json { user } } else { response status=404 "Not found" } } route "POST /api/users" { name = request.body.name email = request.body.email id = database.insert("users", {name, email}) response status=201 json { id } } // Deploy: webbubble api.bub // 100K req/s, 2MB RAM, 500KB binary
REST API Database CRUD

2. Real-Time Chat Server

WebSocket-powered chat with Redis pub/sub. Handles 10K concurrent connections.

// Real-time chat with WebSockets route "/ws/chat" { websocket.on("message", msg => { username = session.user timestamp = time.now() // Broadcast to all connected clients broadcast({username, msg, timestamp}) // Save to database database.insert("messages", {username, msg, timestamp}) }) } route "/api/history" { messages = database.query("SELECT * FROM messages ORDER BY timestamp DESC LIMIT 50") response json { messages } }
WebSockets Real-time Chat

5. GraphQL API

Modern GraphQL API with type safety and automatic documentation.

// GraphQL schema and resolvers type User { id: ID! name: String! email: String! posts: [Post!]! } type Post { id: ID! title: String! content: String! author: User! } query { users: [User!]! user(id: ID!): User posts: [Post!]! } route "/graphql" { result = graphql.execute(request.body.query) response json { result } }
